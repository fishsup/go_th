# 递归

### 算法思想
- 如何将一个问题的规模变小
- 利用从小规模问题中得到的记过, 结合当前的值或情况, 得出最终的结果


### TOP-DOWN 自顶向下
- 把要实现的递归函数看成是已经实现好的
- 直接利用递归函数解决子问题
- 根据子问题的结果及当前情况得出答案
